# ASCII Art Converter

Turn any image into ASCII art!

## How to use

1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the program:
   - **With window (easy!):**
     ```bash
     python ascii_art_converter.py
     ```
     Click "Open Image" and see the magic.
   - **From the console:**
     ```bash
     python ascii_art_converter.py path/to/image.png
     ```
